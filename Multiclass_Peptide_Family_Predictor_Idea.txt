Research Idea: Multi-Class Protein Language Model for Plant Peptide Family Prediction

Title

Development of a Multi-Class Protein Language Model for Plant Peptide
Family Classification

Motivation

Most existing peptide prediction models are binary classifiers that
predict whether a sequence belongs to a single peptide family (e.g.,
defensin vs. non-defensin). Separate models must be trained for each
family, which is inefficient and may produce conflicting predictions.

This project proposes a single multi-class model capable of identifying
the peptide family directly from the amino acid sequence.

Objective

Develop a ProteinBERT or ESM2-based deep learning model that classifies
plant peptide sequences into one of several peptide families.

Example families: - Defensins - Cyclotides - Thionins - Lipid Transfer
Proteins (LTPs) - Snakins - Other / Unknown peptides

Dataset Construction

1.  Collect experimentally validated peptide sequences from databases
    such as:

    -   UniProt
    -   APD3
    -   DBAASP
    -   DRAMP
    -   PlantPepDB
    -   NCBI

2.  Remove duplicate sequences.

3.  Remove sequences with non-standard amino acids.

4.  Reduce redundancy using CD-HIT or MMseqs2 (e.g., 90% identity).

5.  Assign exactly one family label to each sequence.

Example dataset:

  Sequence   Label
  ---------- -----------
  MKLFC…     Defensin
  GIPC…      Cyclotide
  KRCC…      Thionin
  MKLV…      LTP
  KCRR…      Snakin

Model Architecture

Input Protein Sequence ↓ ProteinBERT / ESM2 ↓ Protein Embedding ↓ Dense
Classification Head ↓ Softmax Layer ↓ Predicted Family

Unlike binary prediction, the final layer contains one neuron for each
peptide family.

Training Strategy

-   Train/Test split
-   Five-fold cross-validation
-   Fine-tuning of ProteinBERT or ESM2
-   Early stopping
-   Dropout
-   Learning-rate scheduling
-   Class balancing if necessary

Loss Function

Use CrossEntropyLoss for multi-class classification.

Output

The model returns the probability for each peptide family.

Example:

Defensin 0.91 Cyclotide 0.03 Thionin 0.02 LTP 0.02 Snakin 0.01 Other
0.01

Prediction: Defensin

Evaluation Metrics

-   Accuracy
-   Precision
-   Recall
-   F1-score
-   Matthews Correlation Coefficient (MCC)
-   ROC-AUC (one-vs-rest)
-   Confusion Matrix

Advantages

-   One model predicts multiple peptide families.
-   Eliminates the need for separate binary predictors.
-   Produces a single, unambiguous prediction.
-   Learns similarities and differences among peptide families.
-   Easily extendable by adding new peptide families.

Future Extensions

-   Add additional peptide families.
-   Use embedding visualization (UMAP/t-SNE) to study family separation.
-   Investigate attention maps for biological interpretability.
-   Develop a web server for public use.

Expected Contribution

This work would provide a unified deep learning framework for plant
peptide family identification using protein language models. Compared
with multiple binary classifiers, the proposed system offers a scalable,
efficient, and biologically meaningful solution for peptide family
prediction.
